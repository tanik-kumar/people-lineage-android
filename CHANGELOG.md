# Changelog

## [1.0.0] - 2026-03-22

### Added

- Initial public Android release
- Village-wise landing page and village-level people browsing
- Person records with address, notes, phone, age, and gender fields
- Father, mother, spouse, and child linking
- Family graph viewing
- PDF graph export
- Share, call, and WhatsApp quick actions
- Local mode, cloud sync file mode, and Drive backup mode
