# Security Policy

## Data Sensitivity

This application stores personal and family relationship data. Treat exported files, backups, and shared records as sensitive.

## Reporting a Vulnerability

If you discover a security issue related to local storage, export/share flows, backup files, or sync behavior, report it privately before public disclosure.

Preferred contact:

- GitHub issue only for non-sensitive bugs
- Private contact for security-related issues

## Scope

This project currently focuses on:

- local database safety
- file export and sharing safety
- backup and restore validation
- scoped storage compliance

## Current Notes

- The app is offline-first
- Release APKs should be installed only from trusted sources
- Backup files may contain personal family data and should not be shared casually
