# People Lineage v1.0.0

Initial public release of the Android app.

## Included

- Village-wise landing page
- Add, edit, and delete people records
- Father, mother, spouse, and child relationship linking
- Family hierarchy graph view
- Share person details
- Export family graph as PDF
- Call and WhatsApp quick actions
- Local storage, cloud sync file mode, and Drive backup mode
- Signed release APK attached to this release

## Install

Download the signed APK asset from this release page and install it on a supported Android device.
